jQuery .ajax() is the heavyweight function which is used to communicate with the remote sight. That is, the ajax() function performs the asynchronous HTTP (Ajax) request. The load(), get(), and post() functions all rely on the ajax() function.
jQuery.ajax( url [, settings ] )
url - A string containing the URL to which the request is sent. settings - A set of key/value pairs that configure the Ajax request. All settings are optional.
