While the .load() function provides an easy way to load HTML content from the server, but there are times when you want to obtain JSON, XML, HTML, or simple strings. The .get() function provides more flexibility by allowing the retrieval of these other types of data from the server. Also $getJSON is available for working with JSON data.

jQuery .get() is a global jQuery object (i.e. it has no selector) which is called directly.
.get( url [, data ] [, callback ] [, datatype ] )
url - A string containing the URL to which the request is sent. data - A plain object or string that is sent to the server with the request. callback - A callback function that is executed when the request completes. datatype -
